# LA-ICPMS文档

> **合肥工业大学资源与环境工程学院矿物微区分析实验室**

## LA-ICPMS SpotAnalysis

- 基于MATLAB
- 多实验室支持（中国科学院广州地球化学研究所、中南大学、广州拓岩）

## LA-ICPMS ScanAnalysis

- 基于MATLAB
- 更高的数据准确度
- 更高的空间分辨率
- 多外标无内标计算方案，提高数据准确性
